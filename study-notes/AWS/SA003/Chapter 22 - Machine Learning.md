Amazon SageMaker **
	-> to build and deploy ML algorithm
	How?
		-> Ground Truth: set up and manage labeling jobs for training datasets using active learning and human labeling
		-> Notebook: write algorithm using Jupyter Notebook (Python)
		-> Training: train and tune models
		-> Inference: package and deploy
	Tips:
		-> HA is not enable by default
		-> Elastic Inference EI: reduce cost, speed up through put/decrease latency
		-> Auto-scale
		-> Amazon SageMaker Neo: optimize ML algorithm from SageMaker for specific hardware
