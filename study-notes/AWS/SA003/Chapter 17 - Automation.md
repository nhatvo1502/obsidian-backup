**CloudFormation**

    -> Overview
        -> IAC
        -> immutable infrastructure

    -> Steps
        1. Write code (JSON or YAML)
        2. Deployment
            -> Stack or Stack Set

    -> Stack
        -> Regional
        -> Easily deploy portable stacks into other regions as needed
    
    -> Stack Set
        -> CRUD stacks across multiple accounts and region

    -> Template
        -> JSON
        -> YAML
        -> Template Sections
            -> AWSTemplateFormatVersion (REQUIRED)
            -> Parameters (OPTIONAL)
            -> Mapping (OPTIONAL)
            -> Resources (REQUIRED)
            -> Outputs (OPTIONAL)
            -> Tramsform (OPTIONAL)
